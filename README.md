# Mini Programming Language Interpreter

This project is a mini programming language interpreter implemented in Python. It demonstrates how programming languages work internally by building the core components of an interpreter from scratch, including a lexer, parser, abstract syntax tree (AST), and interpreter.

---

## Features

- **Lexer (Tokenizer):** Converts input text into tokens  
- **Parser:** Builds an Abstract Syntax Tree (AST)  
- **Interpreter:** Evaluates and executes expressions  
- **Error Handling:** Provides clear error messages with line and column information  

---

## Supported Features

### Arithmetic Operations
- `+`, `-`, `*`, `/`

### Unary Operations
- `+x`, `-x`

### Comparison Operators
- `<`, `>`

### Variables
- Assignment: `x = 5`

### Output
- `print(x)`

### Parentheses
- `( ... )`

### Strings 
- String literals: `"hello"`
- String concatenation using `+`

---

## How it Works

1. The lexer converts input text into tokens
2. The parser generates an AST from tokens
3. The interpreter evaluates the AST and produces output
